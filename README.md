# 目黒区 学区域・公園マップ

目黒区の学区域図をもとに、公園の位置を可視化したマップです。

## 概要

目黒区の学区域図をもとに、公園の位置を可視化したマップです。

## 作成背景

子どもの外遊び環境に関心があり、学区ごとにどのような公園があるのかを把握するために作成しました。

目黒区が公開している学区域図などの情報を活用し、公園の位置を重ねて可視化しています。

また、公園は遊び場としてだけでなく、災害時には避難場所や地域コミュニティの拠点としても活用できる地域資源です。

## 工夫した点

- 学区域と公園を同時に確認できるようにした
- 公園の位置を色分けして見やすくした
- 子育てと防災の両方の視点で活用できるようにした

## 考察

作成してみると、学区によって公園の分布状況に違いがあるように見えた。
今回は位置の可視化を中心としたため、公園数や面積までは分析していないが、地域によって利用できる公園環境に差がある可能性を感じた。

また、公園は子どもの遊び場としてだけでなく、災害時の避難場所や地域コミュニティの拠点としての役割も持つため、
公園の配置を把握することは地域の暮らしや安全を考えるうえでも有意義だと感じた。

## 今後の展望

現在は公園を中心に可視化しているが、今後は地域資源マップとして発展させたいと考えている。

具体的には、

- 公園名の追加
- 図書館
- 学校
- トイレ
- 避難所
- 児童館

などの情報を追加し、子育て世帯だけでなく、地域住民や災害時にも活用できるマップを目指したい。

また、オープンデータを活用しながら情報を更新し、将来的にはWebページとして公開することで、地域の情報を分かりやすく共有できる仕組みに発展させたい。
